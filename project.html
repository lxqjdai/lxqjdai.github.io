Intelligent Agent: Pac Man Search
Adapted from the Berkeley Pac-Man Assignments originally created by John DeNero and Dan Klein. This is an AI course project to implement an agent to undertake specifical tasks, as well as to find the path to the end in mazes. What we do are: 1) implement path-search algorithms for DFS, BFS, UCS as well as A*, and 2) design effective heuristics.




3D Deformation transfer
Aim to tansfer the deformation of two 3D face mesh to another face mesh. Utilize PRNet to reconstruct 3D Face mesh from 2D images. Implement a paper from MIT, which design a energy function to transfer the deformation. Our software is novel since it only take 2D images to do this task.
